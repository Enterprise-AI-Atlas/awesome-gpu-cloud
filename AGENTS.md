# AGENTS.md — Awesome GPU Cloud

## Purpose

This repository is a curated registry of GPU cloud providers, platforms, and services for accelerated computing and AI workloads.

## Scope

Cover resources related to renting, managing, and optimizing GPU infrastructure in the cloud:

- GPU cloud providers and marketplaces.
- Bare-metal GPU rental services.
- Spot/preemptible GPU instances.
- Kubernetes and container platforms for GPU workloads in the cloud.
- Cost-optimization and benchmarking resources for GPU cloud.

## Style

- Follow the same format as the other `awesome-*` repositories in the Enterprise AI Atlas organization.
- Label official/vendor resources as `Official` and third-party resources as `Community`.
- Prefer official documentation and links.

## Maintenance

- Run `./scripts/validate-links.sh` before finishing any edit.
- Place entries under the provider or service category they primarily target.

## Important reminders

- Do not provide legal advice.
- Verify that links resolve and are current.
